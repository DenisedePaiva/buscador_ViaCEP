###  &#x1F50D; Buscador ViaCEP

O Buscador ViaCep retorna informações sobre um determinado CEP válido; 

Enviará e receberá dados da API do ViaCEP;

Ao digitar o CEP, retornará dados como: BAIRRO, CIDADE e RUA.
